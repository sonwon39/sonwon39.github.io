테이프를 이용한 박리 공정으로 분리한 얇은 MoS2, WSe2 등 2D 반도체의 전압·전류 특성을 측정해 다이오드 거동을 분석하기 위해 만든 계측 자동화 앱입니다. pyvisa로 Keithley source-measure unit을 제어하여 voltage sweep을 자동 실행하고, 결과를 실시간 I-V 곡선으로 그래프에 띄웁니다. Keithley 2420과 2635B 두 모델에서 사용 가능하도록 하기 위해 모두 동일한 인터페이스로 추상화했습니다.

측정 자동화 + 실시간 시각화

핵심 기능

측정 루틴은 QThread로 분리해 UI 응답성을 유지하고, linear sweep · bias · step · dual sweep 네 가지 모드를 지원합니다. dual sweep으로 정·역방향을 연속 측정하면 hysteresis까지 한 번에 관찰할 수 있어, 2D 물질에서 자주 나타나는 trap 거동을 같은 실행에서 확인합니다.

실시간 그래프는 pyqtgraph로 그리고, 같은 sweep을 선형 스케일과 로그 스케일로 동시에 plot합니다. 측정 중 데이터는 Qt signal로 UI 스레드에 전달되어 한 점이 찍힐 때마다 즉시 화면에 반영됩니다.

voltage·current range, current limit, source delay 같은 SMU 파라미터는 GUI에서 바로 설정할 수 있고, 각 측정 종료 후 결과는 CSV로 저장되어 분석 단계로 넘어갑니다.
