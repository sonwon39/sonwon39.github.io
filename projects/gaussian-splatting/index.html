3D Gaussian Splatting으로 넘어가기 전에, 그 바탕이 되는 아이디어인 목표 원본 이미지를 미분 가능 렌더링으로 학습하는 과정을 2D 이미지에서 먼저 구현해 봤습니다.

2D 가우시안 수백개만으로 목표이미지와 유사하도록 만드는 것이 목표였습니다.

2D 가우시안 이미지 피팅

target → init → trained

가우시안들은 위치·공분산·색·불투명도를 파라미터로 가집니다. 무작위로 가우시안들을 초기화한 init상태에서 시작해서, 렌더링 결과와 원본 이미지의 L2 차이를 손실로 두고 PyTorch로 역전파하며 값을 조금씩 고쳤습니다. 충분히 학습을 시키면 가우시안 덩어리들만으로 원본이 쥬요 특징들이 꽤 살아나느 것을 볼 수 있었습니다.

최적화 과정

학습초기에 흩어져서 형태가 없던 가우시안들이 학습이 진행되면서 자리와 모양, 색을 잡아가며 원본으로 수렴해가다가 일정 시점부터는 가우시안을 추가하는 것을 멈추고 디테일을 높였습니다.
